# Команда агентов — Telegram Copywriter Bot

Архитектура: **динамическая маршрутизация**. Оркестратор принимает все запросы и вызывает только нужных агентов — без лишних токенов на простые задачи.

---

## Структура команды

```
                    ┌─────────────────────┐
                    │    ОРКЕСТРАТОР      │ ← все запросы из Telegram
                    │  claude-opus-4-8    │
                    │  temperature: 0.2   │
                    └──────┬──────────────┘
                           │
          ┌────────────────┼────────────────┐
          │                │                │
          ▼                ▼                ▼
  ┌──────────────┐ ┌──────────────┐ ┌──────────────┐
  │  АРХИТЕКТОР  │ │ДЕКОМПОЗИТОР  │ │ ТЕСТИРОВЩИК  │
  │sonnet-4-6    │ │sonnet-4-6    │ │haiku-4-5     │
  │temp: 0.5     │ │temp: 0.3     │ │temp: 0.1     │
  └──────────────┘ └──────────────┘ └──────────────┘
```

## Агенты

### Оркестратор

| Параметр | Значение |
|---|---|
| **Модель** | `claude-opus-4-8` |
| **Температура** | `0.2` |

**Роль:** Точка входа. Принимает сообщение пользователя, определяет интент (написать пост / редактировать / контент-план / загрузить стиль), координирует других агентов и **пишет финальный контент** в стиле пользователя.

**Инструменты:**
- `route_to_agent(agent, task)` — делегировать задачу
- `read_style_profile()` — загрузить профиль стиля
- `write_post(brief)` — написать пост
- `edit_post(original, instructions)` — отредактировать текст

**Когда вызывается:** всегда, первым.

---

### Архитектор

| Параметр | Значение |
|---|---|
| **Модель** | `claude-sonnet-4-6` |
| **Температура** | `0.5` |

**Роль:** Стратег контента. Создаёт и управляет контент-планом, предлагает темы и серии постов на основе профиля стиля пользователя.

**Инструменты:**
- `read_content_plan()` — прочитать `content_plan.md`
- `write_content_plan(plan)` — сохранить план
- `suggest_topics(style_profile, n)` — предложить N тем

**Когда вызывается:** запросы на просмотр / создание / обновление контент-плана.

---

### Декомпозитор

| Параметр | Значение |
|---|---|
| **Модель** | `claude-sonnet-4-6` |
| **Температура** | `0.3` |

**Роль:** Аналитик стиля и задач. Парсит выгрузки Telegram-канала (MD/JSON), извлекает паттерны стиля (тон, длина, структура, любимые обороты), разбивает сложные задания на подзадачи.

**Инструменты:**
- `parse_telegram_export(file)` — разобрать выгрузку (MD / JSON)
- `extract_style_patterns(posts)` → сохранить `style_profile.json`
- `decompose_task(task)` → список подзадач

**Когда вызывается:** загрузка файла стиля; сложные многошаговые задания.

---

### Тестировщик

| Параметр | Значение |
|---|---|
| **Модель** | `claude-haiku-4-5-20251001` |
| **Температура** | `0.1` |

**Роль:** Контролёр качества. Проверяет написанный контент на соответствие стилю пользователя, краткость, человечность и хлёсткость. Выдаёт оценку и конкретные замечания.

**Инструменты:**
- `check_style(post, style_profile)` → `{score: 0–10, issues: []}`
- `approve_or_reject(score)` → `pass` (≥7) / `fail` + замечания

**Когда вызывается:** после каждой генерации или редактуры поста. При `fail` Оркестратор правит и повторяет — максимум 2 попытки, затем отдаёт лучший вариант с пометкой.

## Маршрутизация по интентам

| Запрос пользователя | Цепочка агентов |
|---|---|
| Написать пост | Оркестратор → Тестировщик |
| Отредактировать пост | Оркестратор → Тестировщик |
| Показать контент-план | Архитектор |
| Создать / обновить план | Архитектор → Декомпозитор |
| Загрузить файл стиля | Декомпозитор |
| Серия постов (сложная задача) | Декомпозитор → Оркестратор → Тестировщик |
